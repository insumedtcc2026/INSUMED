# INSUMED

Sistema web desenvolvido para auxiliar no gerenciamento e acompanhamento do fornecimento de insumos médicos, buscando facilitar a comunicação entre prescritores, profissionais executores e pacientes.

## Sobre o projeto

O INSUMED surgiu a partir do desenvolvimento de Trabalho de Conclusão de Curso (TCC) do Ensino Médio Integrado em Informática para Internet, com o objetivo de propor uma solução digital voltada à organização do processo relacionado à solicitação e acompanhamento de insumos de saúde.

A plataforma busca reduzir falhas de comunicação, melhorar o acesso às informações e tornar o processo mais eficiente para todos os envolvidos.

---

## Objetivos

- Facilitar o acesso às informações sobre insumos médicos;
- Reduzir deslocamentos desnecessários;
- Melhorar a comunicação entre usuários e profissionais;
- Organizar solicitações e acompanhamentos;
- Tornar processos mais rápidos e acessíveis.

---

## Público-alvo

O sistema atende principalmente:

- Pacientes;
- Prescritores;
- Profissionais executores;
- Unidades de saúde.

## Como executar o projeto

Clone o repositório:

```bash
git clone URL_DO_REPOSITORIO
```

Entre na pasta:

```bash
cd INSUMED
```

Instale dependências:

```bash
npm install
```

Execute:

```bash
npm run dev
```

O projeto estará disponível em:

```txt
http://localhost:5173
```
